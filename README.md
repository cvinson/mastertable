## mastertable


[![Build Status][travis-image]][travis-url]

## Install

```bash
$ npm install mastertable
```

## Usage

```js
mastertable = require('mastertable')

mastertable.degreesToPoints(3)
// => 6

mastertable.degreesToPoints(6)
// => 3
```
